# elenext

elenext 是一个基于 vue3 实现的 UI 组价库，组件样式源于 element-ui ，组件设计也会尽量往 element 靠拢，以减少迁移成本

## Docs & Demo

[https://elenext.vercel.app](https://elenext.vercel.app)

## Install

```sh
yarn add elenext
```
